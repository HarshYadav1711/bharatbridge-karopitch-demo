# BharatBridge Karo Pitch Demo

Demo homepage for **Karo Pitch**, a startup pitching platform by KaroStartup. Founders pitch startups to investors; built especially for founders from tier 2 and tier 3 cities. Investors discover promising startups. KaroStartup supports the ecosystem.

## Prerequisites

- Node.js 18+

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project structure

- `app/` – Next.js App Router (layout, page, globals.css)
- `components/` – Shared UI (Header, Footer, Shadcn components under `ui/`)
- `sections/` – Landing page sections (placeholders; replace with Hero, Features, CTA)
- `lib/` – Utilities (e.g. `cn` from Shadcn)
- `styles/` – Extra CSS (optional)
- `public/` – Static assets

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI
- Lucide React
